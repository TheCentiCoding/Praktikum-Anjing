## 1. Permutasi dan Kombinasi
Buatlah program dengan method untuk menghitung nilai permutasi dan kombinasi dari sebuah nilai yang dimasukkan.
```
Program Penghitung Permutasi dan Kombinasi
1. Permutasi
2. Kombinasi
3. Keluar
Masukkan pilihan Anda : 1
Masukkan total objek n = 5
Masukkan jumlah objek yang dipilih r = 2
        n!
nPr = ------ = 20
      (n-r)!
Maka hasil permutasinya adalah = 20
Program Penghitung Permutasi dan Kombinasi
1. Permutasi
2. Kombinasi
3. Keluar
Masukkan pilihan Anda : 2
Masukkan total objek n = 5
Masukkan jumlah objek yang dipilih r = 2
         n!
nCr = -------- = 10
      r!(n-r)!
Maka hasil kombinasinya adalah = 10
Program Penghitung Permutasi dan Kombinasi
1. Permutasi
2. Kombinasi
3. Keluar
Masukkan pilihan Anda : 3
Program dihentikan.
```
