# Bab 5 - Method
## 1. Permutasi dan Kombinasi
Buatlah program dengan method untuk menghitung nilai permutasi dan kombinasi dari sebuah nilai yang dimasukkan.
## 2. Bilangan Prima
Program untuk menentukan apakah bilangan tersebut bilangan prima atau bukan dengan menginputkan sebuah bilangan yang di inginkan. Setelah itu program akan menampilkan bilangan prima 1-100.
