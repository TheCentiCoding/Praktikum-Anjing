## 2. Indeks Massa Tubuh
Untuk menentukan kriteria kegemukan, digunakan IMT (Indeks Massa Tubuh), yang bisa dihitung menggunakan rumus:
> IMT = b / t2
> b = berat badan (kg)
> t = tinggi badan (m)
Kriteria untuk nilai IMT ditabelkan sebagai berikut:
| Nilai IMT | Kriteria |
|----------|----------|
| IMT ≤ 18,5  | Kurus  |
| 18,5 < IMT ≤ 25  | Normal  |
| 25 < IMT ≤ 30  | Gemuk  |
| IMT > 30  | Kegemukan  |
